# zpFolio

Personal portfolio built with Vue 3, TypeScript, GSAP and Lenis.

## Tech Stack

- **Vue 3** + TypeScript
- **GSAP** — scroll-triggered animations & typing effect
- **Lenis** — smooth scrolling
- **Vite** — build tool

## Getting Started

```bash
# install dependencies
pnpm install

# run dev server
pnpm dev

# build for production
pnpm build

# preview production build
pnpm preview
```
